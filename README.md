# Timesheet App in ROR

Business Timesheet application built in ROR

#Ingredients Used
* Rails 4/5
* JQuery Plugins
* HAML
* SASS/SCSS
* Custom Bootstrap - Gentelella
* Devise
* CanCanCan
* High Voltage

## Installation
1. Download the zipped folder
2. Navigate to the project folder
3. `bundle install`
4. `rails s`
5. Open `http://localhost:3000`
